# YouTube Music Desktop

A simple desktop app for YouTube Music with built-in ad blocking.

## What is this?

This is a lightweight desktop application that lets you enjoy YouTube Music without ads. It's built with Electron and includes an ad blocker for a better listening experience.

## Getting Started

### Installation

First, install the dependencies:

```bash
npm install
```

### Running the App

To start the app:

```bash
npm start
```

## How to Use

- The app opens YouTube Music in a dedicated window
- Close the window to minimize to system tray
- Right-click the tray icon to show the app or quit completely
- Your login and preferences are saved automatically
